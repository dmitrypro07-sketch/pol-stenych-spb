# ПолСтеныч — полусухая стяжка пола в СПб и ЛО

Одностраничный лендинг для **Никиты** — частного мастера по механизированной полусухой стяжке пола в Санкт-Петербурге и Ленинградской области. Работает с 2015 года, бригада 4 человека, материалы: евроцемент М-500 + армирующее фиброволокно.

**Домен:** полстеныч.рф (планируется)
**Телефон:** +7 938 808 72 66

## Стек
- HTML5 + CSS3 + Vanilla JS — всё в одном файле
- Google Fonts: Bebas Neue, DM Sans, DM Mono
- Видео-фон в hero (H.264, MP4)
- Без сборщика, без зависимостей

## Секции
1. Hero (видео-фон + лазерная стрелка-прицел)
2. О мастере
3. Trust-bar
4. Технология полусухой стяжки (4 шага)
5. Преимущества
6. Как работаем (5 шагов)
7. Калькулятор стоимости
8. Портфолио (6 фото работ)
9. Отзывы с Авито (карусель)
10. Что входит в цену
11. FAQ
12. Контакты + форма заявки

## Запуск локально
Просто открой `index.html` в браузере:
```bash
start index.html
```
Либо через live-server:
```bash
npx live-server
```

## Деплой
Статический сайт. Деплой на [Vercel](https://vercel.com) через GitHub integration — push в `main` автодеплоит.
