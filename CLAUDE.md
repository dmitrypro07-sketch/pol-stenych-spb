# CLAUDE.md — ПолСтеныч СПб (Никита)

## Проект
Одностраничный лендинг для **Никиты** — частного мастера по **механизированной полусухой стяжке пола** в СПб и ЛО.
Домен — **полстеныч.рф**.
Всё в одном `index.html` (HTML + CSS + JS inline).

## Заказчик
- **Никита** — частный мастер, 9 лет опыта (с 2015)
- Услуга: **полусухая стяжка пола** (и только она)
- Регион: Санкт-Петербург + Ленинградская область
- Бригада: 4 человека, постоянный состав
- Материалы: евроцемент М-500 + фиброволокно
- Работает с физиками и юриками, закрывающие документы

## Стек
- HTML5 / CSS3 / Vanilla JS (один файл)
- Google Fonts: Bebas Neue, DM Sans, DM Mono
- Без npm и сборщиков — просто открываем index.html
- Референс: `C:\Code\VSCode\nalivnye_poly_msk` (почти такой же проект)

## Правила разработки
- **Не менять палитру** без явной просьбы Михалыча
- **Не подключать** внешние зависимости (npm, фреймворки)
- Весь JS — ванильный
- Услуга — **только полусухая стяжка пола**, ничего больше (стены, плитка и пр. не добавляем)
- Тон текстов — спокойный, экспертный, без пафоса («свой мастер, а не диспетчер»)
- Все тексты — на русском
- Адаптив обязателен — вплоть до 480px
- Бренд пишется как **ПолСтеныч** (с заглавной С), домен — `полстеныч.рф`
- В заголовках и текстах использовать факты Никиты:
  - «с 2015 года» / «9 лет»
  - «бригада из 4 человек»
  - «евроцемент М-500 и фиброволокно»
  - «фиксированная цена в договоре»
  - «весь мусор упаковывается и выносится»
